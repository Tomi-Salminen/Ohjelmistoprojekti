# Ohjelmistoprojekti
## Backend
In root folder  
```
    docker compose up -d
```

Create .env file in backend folder:  
```
    JWT_KEY="whatEverYouWant"
```

In backend folder:  
```
    npm install
    npm run dev
```

## Frontend
Create .env file in frontend folder:  
```
    VITE_API_URL=http://localhost:3001
```

In frontend folder:  
```
    npm install
    npm run dev
```
